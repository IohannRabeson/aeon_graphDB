## WhiteDB:

#### Download:
Download "whitedb" here : http://whitedb.org/download.html

#### Install:
Then to install type: (more details here: http://whitedb.org/install.html)
```bash
./configure
make
make install
```

## Initialisation:
```bash
./Init.sh
```

## Run the test:

```bash
make
./aeon_graphDB
```
